Three layers of intelligence

📐

Rule-Based Engine

Hard-coded thresholds for altitude jumps, impossible speeds, squawk 7500/7600/7700, and MLAT inconsistencies. Zero-latency first pass.

Layer 1
🌲

Isolation Forest

Unsupervised ML model isolating anomalous behaviour by scoring feature vectors across velocity, heading, altitude, and climb rate.

Layer 2
🧠

Deep Autoencoder

LSTM-based autoencoder trained on benign ADS-B sequences. High reconstruction error signals novel spoofing patterns.

// CAPABILITIES

Built for production

🛰️

OpenSky Network Integration

Direct REST API pulls over Indian airspace bounding box. Configurable polling interval for live vs. batch modes.

⚡

Sub-second Inference

All three pipeline layers run sequentially in under 400 ms per batch, enabling real-time detection without queuing lag.

📊

Feature Engineering

Velocity deltas, heading consistency, barometric vs. geometric altitude divergence, and temporal sequence embedding.

🔔

Severity Triage

Three-tier severity — INFO, WARN, CRITICAL — with per-anomaly drill-down and confidence scores.

🗂️

Structured Logging

All detections persisted to structured JSON logs with ICAO24, callsign, detection layer, score, and UTC timestamp.

🧩

Modular Architecture

Each layer is independently swappable. Drop in XGBoost, VAE, or custom rules via a clean detector interface.
