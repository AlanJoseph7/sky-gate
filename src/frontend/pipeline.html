// DETECTION PIPELINE

Three layers of intelligence

From hard physics rules to unsupervised ML to deep reconstruction — each layer catches what the others miss, combined by dynamic F1-weighted voting.

// PIPELINE STAGES
01

📡 Data Ingestion

generator.py

Pulls live ADS-B transponder data from OpenSky Network and adsb.lol APIs every 30 seconds over Indian airspace. Synthetic anomalies are injected into real flight data for model training and evaluation — covering all 6 anomaly classes.

02

⚙️ Feature Engineering

features.py

Computes derived features from per-aircraft history buffers: climb rate, lateral acceleration, heading change rate, distance-per-step, and barometric vs. geometric altitude divergence. Uses compute_features_live() with a rolling aircraft buffer to avoid NaN issues from single-snapshot data.

03

📐 Rule-Based Engine

rules.py

Physics-threshold first pass — flags altitude jumps (>5000ft/step), teleportation (>50km/step), impossible speeds (>1000 kts), and emergency squawks (7500/7600/7700). Zero latency. Highest precision of all three layers.

0.91Precision
0.76Recall
0.83F1 Score
04

🌲 Isolation Forest

isolation_forest.py

Unsupervised ML model trained only on normal flight data. Scores feature vectors by how quickly a tree can isolate them — anomalous patterns are isolated in fewer splits. Catches velocity, heading, and altitude outliers that rules might miss.

0.46Precision
0.67Recall
0.55F1 Score
05

🧠 Feedforward Autoencoder

lstm_model.py

Deep neural network trained to reconstruct normal ADS-B feature vectors. High reconstruction error signals novel spoofing patterns — gradual speed drift, altitude oscillation, heading zigzag. Bottleneck of 2 neurons forces compression. Anomaly/normal ratio: 1.55x.

0.56Precision
0.46Recall
0.50F1 Score
06

⚖️ Weighted Ensemble

detect.py

Combines all three detectors using dynamic F1-based weights. If any detector degrades, it automatically loses voting power. Best threshold determined by sweeping [0.1, 1.0] and maximising F1. Self-correcting and retraining-aware.

0.71Precision
0.83Recall
0.77F1 Score

# Dynamic F1-weighted ensemble — detect.py
weight_rule = F1_rule / (F1_rule + F1_IF + F1_AE)
weight_IF = F1_IF / (F1_rule + F1_IF + F1_AE)
weight_AE = F1_AE / (F1_rule + F1_IF + F1_AE)

final_score = (
  weight_rule * rule_anomaly +
  weight_IF * if_anomaly +
  weight_AE * lstm_anomaly
)

# Best threshold found via F1 sweep over [0.1, 1.0]
final_anomaly = final_score >= best_threshold
